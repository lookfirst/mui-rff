# Code of Conduct

We want this project to be open, useful, and respectful.

Everyone who participates here is expected to act professionally and treat other people with respect.

## Expected Behavior

- Be polite and constructive.
- Be respectful when you disagree.
- Accept feedback in good faith.
- Focus on improving the project.
- Be considerate of other people's time and experience.

## Unacceptable Behavior

- Harassment, threats, or intimidation.
- Insults, personal attacks, or abusive language.
- Sexualized language, imagery, or unwanted attention.
- Trolling or deliberately disruptive behavior.
- Sharing someone else's private information without permission.
- Any other conduct that would be inappropriate in a professional setting.

## Maintainer Responsibilities

Maintainers may edit, remove, or reject comments, issues, pull requests, commits, or other contributions that do not follow this code of conduct.

Maintainers may also temporarily or permanently restrict participation when necessary to protect the project and its community.

## Scope

This code of conduct applies in project spaces and in public spaces when someone is representing the project.

Examples include using an official project email address, posting from an official account, or acting as a project representative at an event.

## Reporting

To report unacceptable behavior, please file a GitHub issue in this repository so it gets our attention. The report can be anonymous, and it does not need to include every detail up front.

Reports will be reviewed and handled as fairly and confidentially as possible.
